# Projeto Laravel (PHP 8.3) + Docker

Aplicação web construída com **PHP 8.3** e **Laravel**, empacotada e executada via **Docker** para facilitar o setup em qualquer ambiente.

---

## Requisitos

-   **Docker** e **Docker Compose**

> Não é necessário instalar PHP/Composer localmente para rodar o projeto, pois o ambiente é provido via Docker.

---

## Como rodar o projeto

### Windows

No PowerShell ou CMD, na raiz do projeto:

```bash
windows .\docker-setup.bat
```

###Linux

No terminal, na raiz do projeto:

```bash
bash docker-setup.sh
```
